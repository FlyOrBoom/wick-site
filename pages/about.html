What is the Wick Editor?

The Wick Editor is a free and open-source tool for creating games, animations, and everything in-between. It's designed to be the most accessible tool for creating multimedia projects.

The Wick Editor is a hybrid of an animation tool and a coding environment, heavily inspired by similar tools such as Flash, HyperCard, and Scratch. It was developed in response to a growing need for such a tool for the modern web. You can read more about the creation of the Wick Editor here.

Philosophy

The Wick Editor is free software. This means software that respects users' freedom and community. You can run, distribute, change, and redistribute the software as you wish. You can learn more about free software at the Free Software Foundation.

The Wick Editor is also open-source, which means the source code is made freely available. You can view the codebase and even help to build Wick on GitHub. Wick itself wouldn't be possible without the work being done on many open-source projects, including fabric.js, paper.js, pixi.js, Ace, tween.js, JS Beautifier, bowser, gif.js, howler.js, potrace, random-js, and Spectrum.

The Wick Editor team believes that independent artists and creators deserve their own tools whose purpose and direction they control. We encourage users to study and modify The Wick Editor's source code and become part of the Wick Editor community.

The Wick Editor follows in the footsteps of other projects such as Wikipedia and The Internet Archive that focus on providing a social good and promoting a healthy Internet.
